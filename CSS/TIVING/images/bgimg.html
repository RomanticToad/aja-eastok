자주 찾는 질문

  • TVING 회원가입은 TVING 계정, SNS 연동 계정, CJ ONE 통합 계정으로 가입이 가능합니다.

    * SNS 연동 계정 종류 : Facebook, Twitter, Naver, Kakao, Apple

     

    ■ 회원가입 방법
    1) PC (WEB)
    ① 티빙 WEB 접속
    ② [티빙 시작하기] 클릭
    ③ 가입할 계정 유형 선택 (TVING, SNS, CJ ONE 중 유형 선택)
    ④ 회원가입하기

     

    2) MOBILE (APP)
    ① 티빙 APP 접속
    ② [티빙 시작하기] 클릭
    ③ 가입할 계정 유형 선택 (TVING, SNS, CJ ONE 중 유형 선택)
    ④ 회원가입하기

  • TVING WEB과 APP은 아래와 같은 방법으로 로그인이 가능합니다.

     

    ■ TVING 로그인 방법
    1) 티빙 WEB/APP 접속
    2) '티빙 시작하기' 버튼 클릭
    3) 계정 선택화면에서 회원가입하신 계정 유형 선택
    4) 아이디, 비밀번호 입력 후 '로그인하기' 버튼 클릭

     

    혹시 일치하는 회원정보가 없다는 알림 메시지가 나오신다면 아래 사항을 확인하여 주세요.

     

    ■ TVING 로그인 안될 시 조치 방법
    1) WEB 브라우저 또는 APP 좌측 상단의 '뒤로가기'를 클릭하여 계정 유형 선택 화면으로 이동
    2) 회원가입한 유형을 다시 확인하여 선택
    - CJ ONE 통합회원이신 경우 'CJ ONE으로 시작하기' 선택 (제일 밑에 위치)
    - TVING ID로 가입하신 경우 'TVING ID로 시작하기' 선택
    - 네이버, 카카오 등 SNS 연동 계정으로 가입하신 경우 '각 SNS로 시작하기' 선택
    3) 아이디, 비밀번호 입력 후 '로그인하기' 버튼 클릭하여 로그인

     

    * 'TVING ID'로 로그인 시도하셨는데 일치하는 회원정보가 없다면 먼저 'CJ ONE으로 시작하기'를 선택하여 확인을 부탁드립니다.
    * 아이디가 이메일 형태의 계정인데 'TVING ID'로 로그인이 안되시는 경우 SNS 연동 회원일 수 있으며, 네이버, 카카오 등 '각 SNS로 시작하기'를 선택하여 확인을 부탁드립니다. 
    * 계정 유형을 맞게 선택하신 경우 '아이디 찾기', '비밀번호 찾기'를 진행해 주세요.

     

    지속해서 로그인이 되지 않으시는 경우 1:1 게시판 문의 또는 tving@cj.net 으로 문의해 주시면,
    신속하게 가입하신 계정 확인하여 답변드리겠습니다.

  • APP과 PC WEB 계정 선택 화면에서 최근에 마지막으로 로그인하신 계정을 확인하실 수 있습니다.
    최근 로그인하신 계정을 선택하여 주시고, 혹시라도 일치하는 회원정보가 없다는 알림 메시지가 나온다면 아래 사항을 확인하여 주세요.

     

    ■ TVING 계정 확인 방법
    1) 계정 선택 화면 가장 하단 '아이디 찾기' 클릭
    2) 본인인증으로 찾기 > 본인인증하기
    3) 가입한 계정 ID들 안내  
    4) 계정 ID 옆 (유료)로 표기된 계정으로 로그인

     

    ※ 이메일로 찾기 시 확인되지 않으니, 반드시 본인인증으로 찾기로 이용 계정 확인해주세요.
    ※ SNS 회원은 해당 SNS 아이디가 아닌 티빙 가입 시 등록한 이메일을 알려드립니다.
    ※ 본인인증이 완료된 계정만 확인이 가능합니다.
    ※ 네이버플러스멤버십 이용권 계정은 본인인증찾기로 확인되지 않습니다.

     

    지속해서 로그인이 되지 않으시는 경우, 1:1 게시판 문의 또는 tving@cj.net 으로 문의해 주시면,
    신속하게 가입하신 계정 확인하여 답변드리겠습니다.

  • 유료 가입한 계정을 찾고 싶을 때,
    아래 방법으로 계정을 찾으신 후 계정 유형을 선택하여 로그인하여 주시기 바랍니다.

     

    ■ 이용 계정 확인 방법
    ① 티빙 WEB/APP 접속
    ② [티빙 시작하기] 버튼 클릭
    ③ 가장 하단 [아이디 찾기] 클릭
    ④ 본인인증으로 찾기 → [본인인증하기] 클릭
    ⑤ 가입한 계정 ID들 안내
    ⑥ 계정 ID 옆 (유료)로 표기된 계정으로 로그인

     

    ※ 이메일로 찾기 시 확인되지 않으니, 반드시 본인인증으로 찾기로 이용 계정 확인해주세요.
    ※ SNS 회원은 해당 SNS 아이디가 아닌 티빙 가입 시 등록한 이메일을 알려드립니다.
    ※ 본인인증이 완료된 계정만 확인이 가능합니다.
    ※ 네이버플러스멤버십 이용권 계정은 본인인증찾기로 확인되지 않습니다.

    보다 자세한 확인이 필요한 경우, [1:1 게시판 문의] 또는 [tving@cj.net]로 문의해 주시면 가입하신 계정 확인하여 답변드리겠습니다.

  • 로그인이 되지 않으시는 경우 아래 티빙 대표메일로 이메일 문의를 남겨주시면 확인 후 답변드리겠습니다. 
    - 티빙 대표메일 : tving@cj.net

     

    문의 내용에 발생 증상 외 기기 모델명, OS 버전, 브라우저, 네트워크 등 상세 정보를 남겨주시면 더욱 빠른 조치가 가능하오니 이용에 참고 부탁드립니다.

  • 티빙 라이트 이용권은 KT 통신사 요금제 혜택으로 제공되는 이용권으로 실시간 채널, 일부 TV프로그램, 일부 영화 시청이 가능합니다.
    티빙 서비스 내에서 별도 구매는 어려우시며, 티빙 라이트 이용권의 상세 이용 권한은 다음과 같습니다.

     

    ■ 티빙 라이트 이용권 안내
    ① 실시간 채널+TV프로그램 + T ONLY 방송/영화 + TVING ORIGINAL 영화, 720p HD 화질 이용 가능
    ② 동시 시청 1명, 모바일/태블릿 + PC + 크롬캐스트 이용 가능 (TV 기기 이용 불가)

     

    * 티빙 라이트 이용권은 TVING ORIGINAL 방송, 파라마운트 플러스, 월정액 영화 콘텐츠가 제공되지 않습니다.
    * 단, 2022년 12월 1일 ~ 2023년 6월 30일까지 TVING ORIGINAL 방송 콘텐츠가 한시적 특별 제공됩니다.
    * 티빙 라이트 이용권을 이용하시는 경우 이용권 업그레이드는 제공되지 않습니다.
    * 티빙 라이트 이용권 이용 중 다른 이용권으로 변경을 원하시는 경우 다른 계정으로 구매하시거나, 이용권 만료 후 구매하셔야 합니다.

  • T ORIGINAL은 TVING에서 자체 제작한 콘텐츠를 의미합니다.

    주요 콘텐츠는 많은 고객님들께서 사랑해주셨던 [환승연애2], [술꾼도시 여자들2], [아일랜드], [만찢남] 등이 있으며, T ORIGINAL 콘텐츠의 시즌2 및 신규 콘텐츠를 준비 중에 있으니 앞으로 더 재미있는 티빙을 기대해주시고 많은 이용 부탁드립니다.

  • 티빙 라이트 이용권은 KT 통신사 요금제 혜택으로 제공되는 이용권으로 관련 요금제를 사용하고 계시다면 제공받으실 수 있습니다.

    티빙 라이트 이용권의 경우 티빙에서 별도 구매나 등록하는 방법이 아닌 KT 통신사 사이트에서 등록이 필요하신 점 참고하시어 이용에 불편 없으시길 바랍니다.

  • 만 19세 이상 시청 가능한 콘텐츠는 시청하시려는 프로필의 성인인증이 되어야만 시청 가능합니다.
    성인 인증은 아래와 같이 진행 가능합니다.

     

    ■ 성인인증 방법 안내
    1) PC (WEB)
    ① 티빙 WEB 로그인
    ② 만 19세 이상 등급 콘텐츠 접근
    ③ [휴대폰 본인 인증] 또는 [I-PIN 본인 인증]으로 성인인증 진행

     

    2) MOBILE (APP)
    ① 티빙 APP 로그인
    ② 우측 상단 [프로필 아이콘] 클릭
    ③ 우측 상단 [톱니바퀴 아이콘] 클릭
    ④ [프로필 성인인증] 메뉴 클릭
    ③ [휴대폰 본인 인증]으로 성인인증 진행

    인증 유효기간은 앱 설정 → [프로필 성인인증] 메뉴에서 확인하실 수 있습니다.

     

    * 성인인증 메뉴가 보이지 않으실 경우 앱을 최신 버전으로 업데이트 해주시기 바랍니다. 
    * 성인인증은 계정별 인증이 아닌 프로필별 인증이 필요합니다.
